Javascript para ingeniería frontend

¿Qué es?

Este curso se enfoca en aclararlos fundamentos del lenguaje Javascript y las herramientas fundamentales que debería manejar cualquier desarrollador para aplicar un enfoque de ingeniería a las aplicaciones.

Beneficios

  • Entender cómo se ejecuta el código Javascript en el navegador.
  • Dominar las funciones y los callbacks.
  • Saber decidir y fundamentar entre diferentes frameworks para cada proyecto.

Modalidad Clase personalizada 1 a 1 a través de una videollamada

Planificación

Día 1 “Fundamentos del Framework”

  • Operadores de asignación, precedencia y operadores, tipos de datos, niveles de acceso.
  • La importancia de las funciones.
  • Ambito léxico y Closures.
  • El estándar Ecmascript.

Día 2 “Más sobre los fundamentos de Javascript”

  • Arreglos: Ciclos funcionales vs Ciclos void y enfoques de programación Callbacks.
  • Javascript NO es asíncrono: ¿Qué es el call stack, el call Queue y el motor v8?

Día 3 “Taller Práctico: Aplicando la Teoría contra pruebas unitarias”

  • Desarrollar ejercicios contra una base de pruebas
  • Fundamentos de la programación Funcional
  • Fundamentos de la programación Basada en Objetos
  • Resolución de ejercicios usando programación funcional vs orientada a objetos

Día 4 “Web APIS y más Teoría de Javascript”

  • DOM
  • LocalStorage, SessionStorage, Cookies y Service Workers
  • Compatibilidad entre navegadores
  • Herramientas

Día 5 “Trabajando con DOM y HTML”

  • Aplicar lo aprendido para implementar un requerimiento que involucra interacción de un usuario

Día 6 “Optimización y aplicación de conceptos en un Trabajo Final”

  • ¿Cuando Optimizar?
  • Tips para mejorar el rendimiento
  • Herramientas para medir y cómo medir
  • Diferencias entre entornos para Frontend y el papel de Node.js
